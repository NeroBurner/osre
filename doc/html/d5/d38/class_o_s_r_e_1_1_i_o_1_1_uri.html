Detailed Description

This class represents a URI ( see http://en.wikipedia.org/wiki/Uniform_Resource_Identifier ). You can use it to describe the location of resources like:

file://c:/texture.jpg

For this example the schema is: file. For this example the path is c:/. For this example the resource name is texture.jpg.
