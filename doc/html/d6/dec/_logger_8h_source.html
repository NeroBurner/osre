OSRE::Common::AbstractLogStream
This pure abstract interface class declares the protocol for log-stream implementation. You can use log streams to pipes logs to you preferred output device like a simple text-file or a log window in a GUI application. Implement the write message and attach a valid instance to the logger itself to use your own implementation.
